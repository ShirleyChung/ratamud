# Person Descriptions 資料檔說明

## 檔案位置
worlds/person_descriptions.json

## 用途
此檔案包含了角色描述系統所使用的所有描述詞彙，包括：
- 外貌描述（1-100 分，每一分都有對應的形容詞）
- 力量描述（5 個等級）
- 體格描述（4 個等級）
- 健康狀態描述（5 個等級）

## 為什麼放在 worlds 資料夾？
1. 這是遊戲內容資料，不是程式碼
2. 可以讓玩家或管理員輕鬆修改描述詞彙
3. 不同的世界可以有不同的描述風格（未來可擴展）
4. 方便版本控制和備份

## 修改方式
直接編輯 person_descriptions.json 檔案，重啟遊戲即可生效。
