========================================
3. 技術亮點
========================================

1. 多層資料夾結構
   - worlds/ 存放所有世界
   - 每個世界一個資料夾
   - 支持多世界擴展

2. JSON 持久化
   - 使用 serde_json 序列化
   - 人類可讀格式
   - 容易備份和編輯

3. Observable Pattern
   - 統一的側邊面板展示方式
   - 支持任何實現 Observable 的物件
   - 易於擴展新的可觀察物件

4. 自動管理
   - 啟動時自動建立資料夾
   - 加載時自動同步元數據
   - 添加地圖時自動更新
