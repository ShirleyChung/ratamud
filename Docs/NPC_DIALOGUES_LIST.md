## 🎮 如何測試

### 快速測試指令
```bash
# 啟動遊戲
cargo run --release --bin main

# 訪問最近的 NPC（李老闆在玩家出生點附近）
goto 51 50

# 查看所有 NPC 列表
npcs
```

### 測試不同積極度
```bash
# 100% - 商人（一定會說話）
goto 66 63

# 80% - 醫生
goto 5 0

# 70% - 農夫 wang
goto 48 51

# 60% - 工人
goto 73 67
```

### 運行自動化測試
```bash
./test_all_npc_dialogues.sh
```

## 💡 設計理念

1. **商人最熱情**：積極度 90-100%，因為他們需要招攬生意
2. **醫生較友善**：積極度 80-85%，關心他人健康
3. **農夫樸實**：積極度 70%，話不多但友善
4. **工人忙碌**：積極度 60-75%，專注工作但偶爾打招呼
5. **工程師專注**：積極度 65%，沉浸在項目中

這樣的設計讓每個 NPC 都有自己的性格特點！

## 🔄 修改台詞

如果想修改某個 NPC 的台詞或積極度：

```bash
# 修改台詞
setdialogue <NPC名稱> 見面 <新台詞>

# 修改積極度
seteagerness <NPC名稱> <0-100>
```

範例：
```bash
setdialogue 李老闆 見面 您好！今天想買點什麼呢？
seteagerness 李老闆 50
```
