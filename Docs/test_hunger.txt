測試飢餓與進食系統：

1. 查看初始 HP：
   status 或 i
   → 可看到目前 HP 值

2. 等待時間流逝（每小時會扣 100 HP）：
   → 系統會自動扣除 HP
   → 再次查看 status 確認 HP 減少

3. 創建食物：
   create item apple
   或 create item 蘋果
   → 物品會掉落在地上

4. 撿起食物：
   get apple
   或 get 蘋果
   → 食物會進入背包

5. 吃食物回復 HP：
   eat apple
   或 eat 蘋果
   → 消耗 1 個蘋果
   → 回復 300 HP
   → 顯示當前 HP

6. 查看其他食物的回復值：
   - 蘋果 (apple): 回復 300 HP
   - 麵包 (bread): 回復 500 HP
   - 乾肉 (jerky): 回復 800 HP
   - 漿果 (berry): 回復 200 HP

注意：
- 每小時自動扣除 100 HP
- 只有食物才能用 eat 指令
- 吃非食物會提示錯誤
- HP 不會低於 0
