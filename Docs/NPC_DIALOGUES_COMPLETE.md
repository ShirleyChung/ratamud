## 🎨 設計特色

### 1. 職業性格分明
- **商人類**：最熱情（90-100%），主動招攬生意
- **醫生類**：友善關懷（80-85%），關心他人健康
- **農夫類**：樸實話少（70%），但友善親切
- **工人類**：專注工作（60-75%），忙碌但偶爾打招呼
- **工程師類**：沉浸專案（65%），有時會分享工作內容

### 2. 台詞個性化
每個 NPC 的台詞都根據其職業和性格設計：
- 商人強調「商品」、「優惠」
- 醫生關心「健康」、「醫療」
- 農夫談論「收成」、「農田」
- 工人提到「工作」、「修理」
- 工程師說「專案」、「projects」

### 3. 雙語支援
- 中文 NPC 使用中文台詞
- 英文名 NPC 使用英文台詞（merchant, Steve）

## 🎮 立即測試

### 方法一：訪問最近的 NPC
```bash
cargo run --release --bin main
```
然後輸入：
```
goto 51 50  # 訪問李老闆（100%會說話）
```

### 方法二：運行測試腳本
```bash
./test_all_npc_dialogues.sh
```

### 方法三：訪問不同積極度的 NPC
```bash
# 100% - 一定會說話
goto 51 50   # 李老闆

# 70% - 約 7 成機會
goto 48 51   # wang

# 60% - 約 6 成機會  
goto 73 67   # 工人
```
