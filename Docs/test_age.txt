測試年齡追蹤系統：

1. 查看自己的年齡：
   status 或 i
   → 會顯示 "存在時間: X秒 (X天X時X分X秒)"

2. 創建 NPC：
   create npc 工人 小明
   → 等待幾秒後再查看

3. 查看 NPC 年齡：
   look 小明
   → 會在側邊面板顯示 NPC 的年齡

4. 創建物品：
   create item 蘋果
   → 物品會掉落在當前位置

5. 查看物品年齡：
   look 或 l
   → 會顯示 "🎁 此處物品: 蘋果 x1 (平均存在: X天X時)"

6. 等待幾秒，重複查看以確認年齡在增長

注意：
- 每秒所有物件的 age 都會 +1
- Person 和 Item 都實現了 TimeUpdatable trait
- 地圖上的物品年齡也會追蹤
