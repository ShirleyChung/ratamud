# SDL 命令的三種語法

## 快速參考

### 語法 1：簡單版（無條件）
```bash
sdl <NPC> <話題> <對話內容>
```
**範例**：
```bash
sdl 商人 見面 哈囉！你好
sdl 櫻花 閒聊 今天天氣真好
```
**適用場景**：簡單、無條件的對話

---

### 語法 2：add 語法（帶條件，舊版）
```bash
sdl <NPC> <話題> add <對話內容> when <條件>
```
**範例**：
```bash
sdl 櫻花 閒聊 add 你真漂亮 when 顏值>80 and 性別=女
sdl 工人 閒聊 add 這工程真難做 when mp<98000
```
**適用場景**：向後兼容，對話內容簡短

---

### 語法 3：set 語法（帶條件，新版）✨
```bash
sdl <NPC> set <話題> when <條件> say <對話內容>
```
**範例**：
```bash
sdl ammy set 閒聊 when 力量>100 and 顏值>80 say 你真是又帥又厲害
sdl 商人 set 閒聊 when 物品數量>5 say 要不要賣些東西給我
sdl 櫻花 set 見面 when mp>500 and 性別=女 say 你好！你看起來精神很好呢
```
**適用場景**：條件複雜時更直觀，條件在前對話在後

---

## 語法 3 的優勢

1. **邏輯更清晰**：`when <條件> say <對話>` 讀起來像自然語言
2. **條件在前**：先看條件，再看對話，符合思考流程
3. **明確分隔**：`say` 關鍵字明確標示對話內容的開始
4. **複雜條件友好**：條件多時不會和對話內容混淆

## 比較範例

同樣的功能，三種寫法：

```bash
# 語法 1（無條件）
sdl 商人 閒聊 歡迎光臨

# 語法 2（有條件）
sdl 商人 閒聊 add 要不要賣些東西給我 when 物品數量>5

# 語法 3（有條件，推薦）
sdl 商人 set 閒聊 when 物品數量>5 say 要不要賣些東西給我
```

## 選擇建議

- 💡 **無條件對話**：用語法 1
- 💡 **有條件 + 短對話**：用語法 2
- 💡 **有條件 + 複雜條件**：用語法 3 ⭐

語法 3 是推薦的新語法，特別適合需要多個條件判斷的情況！
